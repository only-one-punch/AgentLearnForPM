# Agent PM Knowledge Readability Standard

## Goal

Convert each Markdown document from a dense reference note into a study-friendly interview handbook.

The target reader is a strong technical Agent PM preparing for interviews. The document should still be broad and technically useful, but easier to read, review, and speak from memory.

Use `04-Tool-Calling工具调用.md` as the current style sample.

## Required Enhancement

Each document should add a new early section:

```markdown
## 0. 先读这一页
```

This section should include:

1. 三分钟速读
2. 本篇阅读路线
3. PM 决策速查表
4. 学完后你应该能做到

## Required Visual / Structural Aids

Each document should add at least one theme-specific visual aid, preferably Mermaid:

- Concept map
- Workflow / execution flow
- Decision tree
- Evaluation loop
- Risk ladder
- Launch gate flow
- Interview answer flow

The visual should be useful for learning, not decorative.

## Required Ending Section

Before References, add:

```markdown
## 面试卡片与自测
```

or the next appropriate numbered heading.

This section should include:

1. 面试官想考什么
2. 30 秒回答模板
3. 2 分钟回答模板
4. 容易踩坑
5. 读完自测题
6. 掌握标准

## Preserve Existing Content

Do not delete the original core content unless it is clearly duplicated or broken.

Keep references at the end.

Do not turn the document into a shallow summary. The goal is "easier to study", not "shorter and thinner".

## Writing Style

- Prefer Chinese explanations.
- Keep technical terms in English where they are commonly used in interviews.
- Use tables for comparisons and decision criteria.
- Use Mermaid diagrams when they clarify process or structure.
- Use GTM / Sales / Marketing Agent examples where useful.
- Keep PM framing: product decision, user trust, risk, metrics, interview expression.

## Completion Criteria

A completed document should let the reader:

1. Read the first page and understand the module's value quickly.
2. Follow a clear learning path.
3. Use a decision table or decision tree in interviews.
4. Practice with self-test questions.
5. Give 30-second and 2-minute interview answers.
